# Sistema de Streaming de Música

## Descrição

Sistema em Java para cadastrar, listar e buscar músicas por título, artista e gênero, além de exibir estatísticas do acervo musical via terminal.

---

## Funcionalidades

* Cadastrar músicas
* Listar todas as músicas
* Buscar por:

  * Título
  * Artista
  * Gênero
* Exibir estatísticas:

  * Total de músicas
  * Duração total
  * Duração média

---

## Tecnologias utilizadas

* Java
* ArrayList
* Programação Orientada a Objetos (POO)

---

## Como executar

1. Compile o projeto:

```bash
javac StreamingMusica.java
```

2. Execute:

```bash
java StreamingMusica
```
